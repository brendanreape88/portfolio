Hi, I'm Brendan.

I'm a Web Developer and life-long learner based in the LA Area.

ABOUT ME

I'm a fan of building useful things. This passion is one of the primary reasons I've found myself pursuing web development. To me, web development is like the adult equivalent of Legos. Now, instead of sitting on my bedroom floor and building fleets of mini-spacecraft in my bedroom, I'm building robust applications with real-world implications. These days HTML, CSS, JavaScript, and jQuery are my go-to "blocks." I pride myself in my ability to see work as play, and love bringing that mindset into a collaborative space.

Outside of the workplace you can generally find me out in nature, jamming with a band, taking part in group meditations, and working on my online wellness marketplace.

PORTFOLIO

Instagram Clone

This fully responsive Instagram clone includes the basic features of the real-deal app. Users can upload photos as well as leave comments on the photos of others. The app has user authentication built in and a user account is required to do anything beyond browsing the content uploaded by other users.

This full stack app was created with HTML, CSS, JavaScript, Material UI, React, Firebase, Nodejs, Express, Pusher, MongoDB, and Heroku.

Live | GitHub

Covid-19 Tracker

This tracker app takes the essential global statistics on the spread of covid-19 and displays them in a user friendly way. Global data is coupled with dates and displayed in a line chart while country-based data is displayed in a bar chart.

HTML, CSS, JavaScript, Material UI, and React were used for this app.

Live | GitHub

Smoothie.io

This app allows you to build and publish the perfect smoothie by adding items from the smoothie database. You can also favorite other user's smoothies, or even customize them to suit your tastes.

This app was built with HTML, CSS, JavaScript, React, Nodejs, Express, and Postgres. The client is deployed on Vercel, and the backend is deployed on Heroku.

Live | GitHub

ZenPal

Have you been looking for a simple meditation companion app? Well look no further! ZenPal makes it easy to time your daily meditations, and also keep a journal cataloging your experiences.

This app was built with HTML, CSS, JavaScript, React, Nodejs, Express, and Postgres. The client is deployed on Vercel, and the backend is deployed on Heroku.

Live | GitHub
